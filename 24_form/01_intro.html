Form

Form will helps to send client data to the server

name attribute will helps to send form elements to the server

There are tow methods for sending client data to the server

There are tow methods for submitting form data

  1. GET (default)
  2. POST

GET

Get is the default method of form

Get use URL for sending data to the server

Get less secure

Get limitation 2048 char

Get not support file uploading

Get faster then post

Get we can bookmark with data

Get query string ?var=val&var1=val1

ex:file:///c%3A/xampp/htdocs/FrontEnd/23Sept_TTS_8/HTML/24Form/01Intro.html?username=asdfasdf&btn=Submit

secure
