Absolute

Exact

The absolute file path is a full URL (address) of the file path to access an internet file. For example,

file:///C:/xampp/htdocs/FrontEnd/23Sept_TTS_9/HTML/images/bg.jpg

C:\xampp\htdocs\FrontEnd\23Sept_TTS_9\HTML\images

Relative

Executed file and loaded files will be Related with each other

The relative file path describes the path of the file relative to the location of the current web page. For example,

images/filename.extension
